# Digital Health Passport

Digital Health Passport is a mobile application built using React Native, Expo, and Supabase that allows users to securely store and manage personal medical information.

---

## Features

- User Authentication
- Secure Login & Registration
- Medical Information Storage
- Profile Management
- Emergency Contact Information
- Edit Medical Information
- Persistent User Sessions
- Clean Mobile UI

---

## Tech Stack

- React Native
- Expo
- Supabase
- React Navigation

---

## Installation

1. Clone the repository

```bash
git clone https://github.com/XabiChettiar/digital-health-passport.git
```

2. Install dependencies

```bash
npm install
```

3. Create a `.env` file

```env
EXPO_PUBLIC_SUPABASE_URL=your_supabase_url
EXPO_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key
```

4. Start the app

```bash
npx expo start
```
